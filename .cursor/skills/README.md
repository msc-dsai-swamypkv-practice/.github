# Agent skills (organization profile repo)

This org-profile repository is **not** a course codebase. The only bundled content here is a short **org-profile** skill plus this README.

## Mirror

If present under `.github/skills/`, files must be **byte-identical** in `.cursor/skills/` (see `.github/workflows/ci-skills-parity.yml`).

## Bundled skills

| Folder | Purpose |
|--------|---------|
| `org-profile` | Scope for editing `profile/README.md` and org-facing Markdown |

For **course repositories**, use each repo’s own `.github/skills/` and `CLAUDE.md`.
